Premier League and Championship
Transfer Dashboard

I enjoy software development have always enjoyed football but been amazed and the state of the transfer market and every year it seems to get more crazy! So it seemed a good choice of subject to explore this new technology with!

The site has been created to show my ability in website development as I a course I am undertaking has introduced me to, amongst other things, DC.js and the supporting software D3.js and Crossfilter, in addition to Flask, which is a sever side web mirco-framework that this site runs off.

The data for the site is based on data from the website Transfer League which has various stats on associated with Premier and Championship transfers.

Transfer league is a static website with a detailed view, the aim of this site was to take a broader view of the English and Championship transfer data and present an interactive interface where uses could drill down into the data, in essence I wanted to use the some of the data but present a different and interactive perspective.

There are slight differences in the data between transfer league and this site, in that 'undisclosed fees' are counted as 0 on this site, so consider the totals shown on this site, an 'at least' value, rather than an attempt to get an exact value
